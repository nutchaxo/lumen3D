Methods, access, and citation

About This Platform

This platform was developed at IRIBHM, Universite Libre de Bruxelles, to explore confocal microscopy data of mouse embryonic development.

Scientific Scope

Fixed imaging, live imaging, and cell tracking datasets can be explored independently or side by side. Linked datasets expose registration metadata when available.

Methods

Raw ND2, CZI, Imaris, XLSX, and GLB files are converted into web-ready metadata, WebP stacks, compressed JSON tracks, and optimized 3D models.

Data Access

The Download Center lists raw and web-ready files per dataset, with size warnings for very large microscopy sources.

How to Cite

Please cite the IRIBHM microscopy platform, the original experiment, and any publications associated with the dataset. A citation block can be copied from each dataset workspace export.

Scientific Tools

Tools are designed for direct interaction with embryos, cells, cuts, graphs, and exported figures.

Interactive Cuts

Move clipping planes in the 3D embryo and save views as reproducible workspace states.

Cell Analysis

Inspect cell tracks, lineage, speed, displacement, and nearest neighbors from the tracking viewer.

Chart Studio

Build exportable population, velocity, neighbor, and mitosis graphs linked to the timeline.

Figure Builder

Export publication-oriented figures, graph panels, and complete workspace JSON files.
