# Changelog - Plateforme Web

## v0.12.2

### [ADDED]
- **Palette de Couleurs Dynamique :** La grille de couleurs (Channel Panel) a été entièrement réorganisée selon un dégradé logique et fluorescent.
  - Ligne 1 : Couleurs "Néon" primaires et secondaires pures (Rouge, Vert, Orange, Jaune, Violet, Magenta, Bleu Ciel, Cyan, Blanc).
  - Ligne 2 : Dérivés plus clairs (tons pastels/doux).
  - Ligne 3 : Dérivés sombres (ombres et tons profonds).
- **Auto-Détection Fluorescente :** Les couleurs par défaut (fallbacks) de la plateforme Web ont été mises à jour pour utiliser les nouvelles couleurs Flashy (eGFP = Vert pur, DAPI/Hoechst = Bleu cyan lumineux, Pecam/RFP = Magenta/Rouge absolu).
